\# Pipeline CI/CD



\## Descrizione



È stata implementata una pipeline CI tramite GitHub Actions.



\## Funzionamento



La pipeline si attiva automaticamente ad ogni push sui branch:

\- main

\- feature-bacheca



\## Controlli eseguiti



\- Verifica dell'esistenza del file index.html

\- Verifica che il file non sia vuoto



\## Obiettivo



Garantire un controllo automatico di base ad ogni modifica del codice,

simulando un processo professionale di Continuous Integration.
